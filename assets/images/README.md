# Images Directory

Place your images here:

## Required Images:

### Profile Image
- **profile.jpg** (400x400px recommended)
  - Your professional photo
  - Will be displayed in the hero section with a red glow effect
  - Supports: JPG, PNG, or any web image format

### About Image
- **about.jpg** (600x400px recommended)
  - Photo showing your workspace, coding setup, or another professional photo
  - Will be displayed in the About Me section
  - Landscape orientation works best

### Project Images
- **project1.jpg** (800x500px recommended)
- **project2.jpg** (800x500px recommended)
- **project3.jpg** (800x500px recommended)
- **project4.jpg** (800x500px recommended)

## Image Guidelines:

- **Profile Photo**: Square aspect ratio (1:1) works best
- **Project Screenshots**: Use 16:10 or 16:9 aspect ratio
- **Format**: JPG or PNG
- **Optimization**: Compress images for web (use tools like TinyPNG)
- **Size**: Keep file sizes under 500KB for best performance

## Fallback:

If images are not found, the website will:
- Use a generated avatar for profile picture
- Use placeholder images for projects

## Quick Tips:

1. Name your files exactly as shown above
2. Use high-quality, well-lit photos
3. Ensure images are not pixelated
4. Consider using professional photos for your profile
5. For projects, use clear screenshots that showcase your work
